# TrimFast

موقع عربي بسيط لقص الفيديو والملفات الصوتية باستخدام Flask وFFmpeg.

## التشغيل

1. أنشئ بيئة Python افتراضية وفعّلها.
2. ثبّت المتطلبات:

   ```powershell
   pip install -r requirements.txt
   ```

3. شغّل الموقع:

   ```powershell
   python app.py
   ```

4. افتح `http://127.0.0.1:5000`.

نسخة FFmpeg المرفقة داخل `ffmpeg/bin` تُستخدم تلقائيًا. ويمكن بدلًا منها تحديد `FFMPEG_PATH` و`FFPROBE_PATH` في متغيرات البيئة.

## تشغيل TrimFast من الجوال

هذه الطريقة تفتح الموقع على الجوال بينما يعمل Flask وFFmpeg على الكمبيوتر:

1. وصّل الكمبيوتر والجوال بنفس شبكة Wi‑Fi.
2. من PowerShell على الكمبيوتر شغّل:

   ```powershell
   $env:HOST="0.0.0.0"; python app.py
   ```

3. نفّذ `ipconfig` واعثر على عنوان **IPv4 Address** للاتصال الحالي، مثل `192.168.1.10`.
4. افتح في متصفح الجوال: `http://192.168.1.10:5000` مع استبدال الرقم بعنوان الكمبيوتر.
5. إذا ظهر تنبيه Windows Firewall، اسمح لـ Python بالاتصال على شبكة **Private**.

إذا كان الكمبيوتر متوقفًا فلن تتم المعالجة؛ تشغيل Python وFFmpeg مباشرة على الهاتف يحتاج بيئة منفصلة مثل Termux ونسخة FFmpeg متوافقة مع Android.

## الخصوصية والشروط

- صفحة سياسة الخصوصية: `/privacy`.
- صفحة شروط الاستخدام: `/terms`.
- روابط التواصل مع المطور متاحة عبر GitHub وX داخل الصفحتين.
- تُحذف ملفات الوسائط بعد التنزيل أو الإلغاء، وبحد أقصى 12 ساعة للملفات المتروكة.
- يسجل كل مجلد مؤقت وقت إنشائه، ويُنظف التخزين عند تشغيل التطبيق وبصورة دورية أثناء عمله.

## ما يدعمه الموقع

- MP4 وMOV وWEBM وMP3 وWAV.
- معاينة وتحديد وقت البداية والنهاية بدقة أجزاء الثانية.
- اختصارات لمس على الجوال لتثبيت البداية والنهاية عند لحظة التشغيل الحالية.
- قص سريع بالنسخ المباشر دون إعادة ضغط عندما لا تتغير الأبعاد.
- تغيير أبعاد الفيديو إلى 9:16 أو 16:9 أو 1:1 مع معالجة عالية الجودة.
- إلغاء الرفع أو المعالجة من الواجهة.
- حذف ملفات الجلسة تلقائيًا بعد تنزيل النتيجة، مع تنظيف الملفات القديمة.
- شريط تقدم منفصل للرفع والمعالجة، مع دعم السحب والإفلات والوضع الليلي.
- حد رفع افتراضي 2GB يمكن تغييره عبر `MAX_UPLOAD_BYTES`.
